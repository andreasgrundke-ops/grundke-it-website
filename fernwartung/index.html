Fernwartung starten

Ihr IT-Betreuer von Grundke IT-Service wird Ihren PC gleich aus der Ferne unterstützen.

1

Programm herunterladen

Klicken Sie auf den Button und laden Sie das Fernwartungs-Programm herunter.

↓ Fernwartung herunterladen

Windows 10/11 · Portable · Kein Installer

⚠ Browser zeigt „Verdächtiger Download blockiert"?
Das ist normal – die Datei ist noch nicht signiert. Klicken Sie auf „Trotzdem herunterladen" (Chrome / Edge) bzw. „Datei behalten" (Firefox).
2

Programm starten

Starten Sie die heruntergeladene Datei direkt aus dem Downloads-Ordner per Doppelklick.

⚠ „Der Computer wurde durch Windows geschützt"?
Auch das ist normal. So kommen Sie weiter:
  1. Auf „Weitere Informationen" klicken
  2. Dann auf „Trotzdem ausführen" klicken

Eine offizielle Code-Signierung folgt in Kürze – dann entfällt diese Meldung.

3

Bereit!

Das Programm zeigt Ihnen eine Bestätigung, wenn alles bereit ist. Ihr IT-Betreuer verbindet sich dann automatisch.
